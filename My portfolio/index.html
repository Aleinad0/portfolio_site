ETL Pipeline for Daily Weather Data

We were asked to create an ETL project that would incluide extracting data from an API sorce and constructing a seamless pipeline for effficient data processing.

For the project, we decided to use a weather station located at Bromma Airport in Stockholm to source the essential data.

Using Python code, not only did we interact with the API for data retrieval but also performed data cleansing operations before storing it in a database

In this project, my key role within the team was to establish and guarantee the optimal functioning of AIRFLOW, adding a layer of orchestration to our data workflow.

Solar Panels: ROIC Calculator

For our final project, we were asked to delved into sustainability, and develop a project that allinied with the UN's 2030 goals, with the tools we had learned during these 12 weeks.

Our focus? A web application designed to empower users in Sweden by providing estimates on the return on investment for solar panels based on their roof specifications and location.

A cornerstone of our project was the commitment to reliable data. Leveraging an API from the European Commission, we obtained specific irradiation data for locations across Sweden.

Taking it a step further, we applied Machine Learning to refine our model, enhancing the accuracy of electricity price projections over the next 30 years.

In line with an agile methodology, I actively participated in gathering reliable data and contributed to our team's agile workflow.

My contribution extendedd also in crafting our dashboard using Dash, ensuring an intuitive and informative user interface, as well as designing our final presentation.
